Welcome to SOFT's documentation!
================================
You have reached the Quickstart documentation of SOFT, the Synchrotron-detecting Orbit Following
Toolkit. To get the source code of SOFT, check the
`SOFT website <http://ft.nephy.chalmers.se/~hoppe/soft/>`_ or
`SOFT GitHub repository <https://github.com/hoppe93/SOFT/>`_. If you're looking for the gory
mathematical details of how SOFT is implemented, you should look into the
`SOFT Manual <http://ft.nephy.chalmers.se/~hoppe/soft/>`_.
